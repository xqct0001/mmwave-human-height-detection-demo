# 毫米波雷达人体高度检测 MATLAB Demo

这个 demo 用合成的 FMCW 毫米波雷达数据演示一个最小可运行的人体高度检测链路：

- 单人
- 室内
- 静态或低速正站姿
- 已知地面参考
- 俯仰向虚拟阵列

## 运行方式

在 MATLAB 中进入当前目录后直接运行：

```matlab
main
```

脚本会自动完成 3 组标定场景：

- 不同人体身高
- 不同站立距离
- 轻微前后倾斜扰动

然后输出误差表，并绘制一个标称场景的诊断图。

## 文件说明

- `main.m`: 入口脚本，跑全部场景并汇总误差
- `configRadarScene.m`: 雷达参数、处理门限、测试场景配置
- `loadRadarInput.m`: 输入抽象层，目前接的是合成数据
- `simulateHumanRadarFrame.m`: 生成单帧 FMCW 合成雷达数据
- `estimateHeightDemo.m`: 距离处理、俯仰波束形成、x-z 映射、身高估计
- `plotHeightDemoResults.m`: 绘制结果图

## 当前算法

1. 定义雷达安装几何，默认雷达离地 `1.0 m`
2. 构造人体垂向多散射中心模型
3. 叠加少量静态杂波，生成测量帧和背景帧
4. 用 `measurement - background` 做基础静态杂波抑制
5. 做距离向 FFT
6. 对目标距离窗口做俯仰波束形成，得到 `range-angle` 功率图
7. 把检测点映射到 `x-z` 平面
8. 在人体主簇内取垂向散射包络的加权百分位边界，输出人体高度近似值

## 说明与边界

- 这里的“高度检测”定义为垂向散射包络的上下边界，不是工业级绝对测高。
- 这个版本没有接入真实雷达文件；真实数据适配点在 `loadRadarInput.m`。
- 当前没有扩展到多人、行走、复杂姿态、多径强反射或硬件标定误差。
- 为了把演示重点放在高度链路本身，阵列和带宽参数采用了适合验证的配置。
